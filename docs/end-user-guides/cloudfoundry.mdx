# Cloud Foundry (CF)

The guides in our end user guides section will walk you through a complete end-to-end use case for managing Cloud Foundry resources with Crossplane as code.

To realize the use case, you'll:

- [Order] (/docs/crossplane-provider-cloudfoundry/docs/end-user-guides/order-cf-environment) a managed Cloud Foundry Environment in your BTP Subaccount.
- [Install and Configure] (/docs/crossplane-provider-cloudfoundry/docs/end-user-guides/configure-provider-cf) the Cloud Foundry provider to manage Resources in your Cloud Foundy Environment
- [Deploy Workload] (/docs/crossplane-provider-cloudfoundry/docs/end-user-guides/deploy-provider-cf) into your Cloud Foundry Environment
